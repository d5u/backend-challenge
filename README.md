## Description

CRUD application built with [Nest.js](https://github.com/nestjs/nest)

## Installation

```bash
$ yarn install
```

## Setup database & Prisma
```bash
$ yarn db:dev:restart
```
> Note: Download & install [Docker](https://www.docker.com/) if you do not already have it

## Running the app
```bash
$ yarn start:dev
```
